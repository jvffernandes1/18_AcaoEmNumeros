Sobre a plataforma

Esta página apresenta a visão técnica do Ação em Números para documentação acadêmica, incluindo arquitetura, tecnologias, fluxo de acesso, estrutura de dados e fórmulas de projeção usadas no dashboard.

Tecnologias utilizadas

  • Backend: Python com Flask e arquitetura por rotas e serviços
  • Frontend: HTML, CSS e JavaScript com componentes customizados
  • Banco de dados: MongoDB Atlas com PyMongo
  • Dados de mercado: yfinance e Awesome API
  • Testes: pytest com mongomock para cenários de persistência
  • Integração contínua: GitHub Actions

Fluxo de acesso e segurança

  • Cadastro de usuário com hash de senha via Werkzeug
  • Login por e-mail e senha com sessão autenticada
  • Rotas críticas protegidas para usuários logados
  • Preferências de acessibilidade associadas à conta
  • Exclusão de simulações limitada ao próprio usuário

Estrutura do banco de dados

A aplicação usa duas coleções principais no MongoDB.

usuarios

  • user_id, nome, sobrenome, email
  • senha_hash
  • alto_contraste

simulacoes

  • user_id, email, ticker, period
  • aporte_inicial, aporte_mensal, meses
  • retorno_anualizado, valor_projetado
  • criado_em

Arquitetura do sistema

  • Camada de apresentação: templates Jinja e componentes UI
  • Camada de aplicação: rotas Flask com validação de entrada
  • Camada de serviço: coleta de mercado e cálculos financeiros
  • Camada de dados: MongoDB com índices para usuários e simulações

Fórmulas utilizadas no dashboard

Abaixo, detalhamos a fundamentação matemática por trás das projeções de rentabilidade e valor futuro exibidas na plataforma.

1) Retorno anualizado

Calculado a partir da variação percentual no período, normalizada para um ano fiscal (252 dias úteis).

$$\text{retorno}_{\text{período}} = \frac{\text{preço}_{\text{final}}}{\text{preço}_{\text{inicial}}} - 1$$ $$\text{retorno}_{\text{anual}} = (1 + \text{retorno}_{\text{período}})^{\frac{252}{N}} - 1$$
Onde $N$ representa a quantidade de pontos (dias) no histórico analisado.
2) Conversão para taxa mensal

Utilizada para calcular o rendimento sobre os aportes feitos mês a mês.

$$i_{\text{mensal}} = (1 + \text{retorno}_{\text{anual}})^{\frac{1}{12}} - 1$$
3) Projeção do valor futuro (VF)

Composto pela valorização do aporte inicial somada à série de pagamentos mensais (anuidade).

$$VF_{\text{inicial}} = \text{aporte}_{\text{inicial}} \times (1 + i_{\text{mensal}})^{\text{meses}}$$ $$VF_{\text{aportes}} = \text{aporte}_{\text{mensal}} \times \left( \frac{(1 + i_{\text{mensal}})^{\text{meses}} - 1}{i_{\text{mensal}}} \right)$$ $$VF_{\text{total}} = VF_{\text{inicial}} + VF_{\text{aportes}}$$
Caso especial (Taxa Zero)

Se a variação do ativo for nula, a projeção assume o valor nominal investido.

$$VF_{\text{total}} = \text{aporte}_{\text{inicial}} + (\text{aporte}_{\text{mensal}} \times \text{meses})$$
